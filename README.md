# 🐾 Care4Pets – Prototype 2  
A mobile application prototype designed to help pet owners manage their pets’ health, reminders, appointments, and daily care.  
This prototype was developed as part of COMP3074 – Mobile App Development.

---

## 📌 Overview  
Care4Pets provides an easy way for pet owners to track vaccinations, medical records, daily reminders, and general pet care activities.  
The goal of Prototype 2 is to deliver improved UI, screen navigation, and basic front-end logic.
