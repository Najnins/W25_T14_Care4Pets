# Care4Pets 
A native Android application for managing your pets' health records, care instructions, and appointment reminders — all stored privately on your device.

---

## Getting Started

### 1. Donwload the Zip from GIthub

### 2. Open the Project in Android Studio

1. Launch **Android Studio**
2. Click **File -> Open**
3. Navigate to the folder and slect the root `Group14-Capstone-Project-2` folder
4. Wait for Gradle to sync.

## Running the App

1. Use the Android Studio Emulator or
2. Use Android Phone connected via USB debuggin enabled.

## Creating An Account

Care4Pets requires registration before you can use the app. There are no pre-built test acounts - you must create your own.

## How to Register

1. Launch the app and tap through the onboarding screens.
2. Tap **Create Account** o the login screen.
3. Fill form
4. Password must be minimum 8 characters, has a number, has a special character, and an uppercase letter.
